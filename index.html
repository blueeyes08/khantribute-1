How to use Khantribute.

Think of it as “Tinder - for proofreading”

  • Swipe right if translation is good. Swipe left if Bad…
  • Skip to next without voting by pressing the skip button

You can also press the thumbs up/ down buttons to vote.

This is a test version so please click message_alert and send us feedback so we can make the app better

How it works:

Every up or down-vote you and other users make on a sentence( string) is added up.
Every string gets a score based on total votes. Strings with high scores will likely be ready to proofread, and lows scores, probably need editing.
This will make proofreading a lot faster. And ultimately get Khan Academy to more people around the world, faster…
